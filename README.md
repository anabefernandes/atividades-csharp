# Atividades Curso C#

Repositório criado para armazenar as atividades práticas realizadas durante o curso **“C# Essencial 2023 + Bônus LINQ”** da Udemy.  
Com o objetivo de registrar o aprendizado e evolução ao longo do curso.

## Estrutura do Repositório

atividade01/ # Exercícios introdutórios - variáveis, tipos e operadores
atividade02/ # Estruturas de decisão e laços de repetição
ClassesMetodos/ # Atividades do módulo de Classes e Métodos
atividade01.sln # Solução principal do projeto
.gitignore

## Conteúdo do Curso

O curso aborda desde os fundamentos até conceitos intermediários da linguagem C#, incluindo:

- Tipos de dados e operadores  
- Estruturas condicionais e loops  
- Arrays e coleções  
- Classes, objetos e métodos  
- Construtores e sobrecarga  
- Modificadores de acesso  
- Propriedades e métodos estáticos  
- Enumerações e structs  
- Tratamento de exceções  
- LINQ e boas práticas
